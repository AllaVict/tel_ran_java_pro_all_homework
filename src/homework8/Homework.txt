 примеры применения хеш-функций в реальной жизни. 
1.	Проверка целостности данных: хеш-функции используются для проверки целостности файлов, сообщений и других передаваемых данных. Полученный хеш-код сравнивается с оригинальным хеш-кодом, чтобы узнать, были ли изменены данные в процессе передачи.
2.	Аутентификация: хеш-функции используются для хранения паролей в зашифрованном виде. При аутентификации пользовательский пароль хешируется, и хеш-код сравнивается с хеш-кодом, хранящимся в базе данных.
3.	Криптография: хеш-функции используются в криптографических алгоритмах для защиты данных и создания цифровых подписей. Хеш-коды используются для проверки авторства и целостности сообщений и файлов.
4.	Поисковые системы: хеш-функции используются для создания индексов, которые позволяют быстро находить информацию в больших объемах данных, например, в Интернете.
5.	Кэширование: хеш-функции используются для кэширования данных в памяти компьютера или сервера. Полученные хеш-коды используются для быстрого поиска кэшированных данных без необходимости загрузки их с диска.
6.	Блокчейн: хеш-функции используются для создания блоков в блокчейне, чтобы обеспечить безопасность и целостность криптовалютных транзакций и других данных.
